Assumptions

  1. Only one pattern is to be generated: (Triomino Pattern) or (Blinker - Period 2).
  2. The user will not interact directly with the game.
  3. The transformation rate from one pattern to the next is constant.
  4. The grid size, canvas size, number of cells are constant.
  5. Javascript must be enabled for the browser to display the game.
  6. The browser must be up-to-date to support Javascript ES6.

To interact with all the parameters above, this can be done easily in the game.js file
